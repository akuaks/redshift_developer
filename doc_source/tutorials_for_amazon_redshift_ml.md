# Tutorials for Amazon Redshift ML<a name="tutorials_for_amazon_redshift_ml"></a>

You can use Amazon Redshift ML to train machine learning models using SQL statements, and then invoke the models in SQL queries for prediction\. Machine learning in Amazon Redshift trains a model with one SQL command\. Amazon Redshift automatically launches a training job in Amazon SageMaker and generates a model\.
